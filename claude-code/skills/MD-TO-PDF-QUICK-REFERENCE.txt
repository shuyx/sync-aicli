❌ 常见问题 & 解决方案
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1: "Chrome 未找到"
A1: 安装 Chrome
    macOS: brew install --cask google-chrome
    Linux: sudo apt-get install google-chrome-stable

Q2: 中文显示乱码
A2: 确保文件使用 UTF-8 编码，HTML 包含:
    <meta charset="UTF-8">

Q3: 表格被分割到多页
A3: CSS 已包含 page-break-inside: avoid，如仍有问题，
    尝试增加 margin 或减少表格宽度

Q4: PDF 生成太慢
A4: 是正常的（通常 5-30 秒），大文件需要更长时间

Q5: markdown 库未安装
A5: pip install markdown


📋 支持的 Markdown 语法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

标题:          # H1, ## H2, ### H3, #### H4
粗体/斜体:     **粗体**, *斜体*
代码:          `inline` 或 ```python\ncode\n```
列表:          - 项1  或  1. 项1
表格:          | Col1 | Col2 |\n|------|------|\n| A | B |
链接:          [文本](url)
图片:          ![alt](path)
引用:          > 引用文本
分割线:        ---

📊 输出文件大小估算

✅ 工作流建议
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

3. 需要 PDF 时运行: bash md-to-pdf-quickstart.sh report.md

5. PDF 生成完毕，可直接分享或打印
