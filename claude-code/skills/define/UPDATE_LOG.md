# Define Skill 配置更新

**更新日期**: 2026-01-17

## 📝 变更说明

### 之前的配置
- 解释字数限制：300-600 字
- 风格：简洁但全面

### 现在的配置
- 解释字数限制：**无限制** ✅
- 风格：详细、全面、根据需要深入

## ✨ 新特性

现在当你提问概念时，系统会：

1. ✅ **提供详细解释** - 不受字数限制
2. ✅ **根据复杂程度调整** - 简单概念简洁，复杂概念详细
3. ✅ **包含深入分析** - 理论基础、工作原理等
4. ✅ **提供丰富例子** - 具体的应用场景和案例
5. ✅ **详细对比分析** - 与相似概念的详细对比

## 📚 包含的内容

现在的解释性文档可能包括：

- **定义**: 核心概念（可多段落）
- **特性**: 关键特征（详细列表）
- **应用**: 实际用途和场景（具体例子）
- **对比**: 与相似概念的区别（详细对比）
- **相关资源**: 参考链接或相关概念
- **深入分析**: 理论基础、工作原理等
- **实践建议**: 如何应用、最佳实践等
- **案例研究**: 真实场景中的应用例子

## 🎯 使用示例

现在你可以期待这样的回答：

```
你: 什么是 "Sim-to-Real"？

  - 实践建议（如何应用）
  - 相关概念（推荐阅读）
  
  ... 以及任何其他有帮助的内容
```
