## 2. 查找语法错误

```
请帮我确保语法和拼写正确。不要润色文本，如果没有发现错误，请告诉我该段落没有问题。
如果发现语法或拼写错误，请将错误列在两列表格中，第一列放原始文本，
第二列放更正后的文本，并加粗标注修改的关键词。
```

## 3. AIGC 降 AI 味

```
你是一个伪装成人类科研学者的AI，能够将AI生成的文本改写成人类学者常用的表达方式。
请对以下由AI生成的段落进行深度改写，在保持原意的基础上，使其风格更自然、
更符合学术写作的习惯，让人难以分辨其是由AI生成。

改写要求：
1. 打破"第一…第二…第三…"排比结构
2. 混合使用长短句（短句≤10字 + 长句40-50字）
3. 加入"笔者认为"/"不难发现"等学术口语
4. 用具体数据替换空洞论述
5. 适度加入辩证表达
6. 连续3个句子的首字不能相同

待改写段落：
```


## 5. 参考文献格式检查

```
请检查以下参考文献列表是否符合 GB/T 7714-2015 标准格式。
对于不符合的条目，请在表格中列出：编号、原始格式、问题描述、正确格式。

检查项：
- 期刊[J]标识是否正确
- 学位论文[D]标识是否正确
- 英文文献姓名格式是否统一
- DOI 是否完整
- 年卷期页码是否齐全

参考文献列表：
```

## 6. 论文结构评估

```
你是一位资深的学术论文评审专家，请对以下论文目录结构进行评估。
评估维度：
1. 各章之间的逻辑递进关系是否清晰
2. 理论章节是否为后续分析提供了充分支撑
3. 案例分析是否回答了绪论提出的研究问题
4. 结论是否超出了论文的论证范围
5. 是否存在章节内容重叠或缺失
请以表格形式输出评估结果，并给出具体改进建议。

论文目录：
```

## 7. 论文查重 AI 检测预检

```
请分析以下段落是否可能被 AI 内容检测系统（如 Turnitin AI、知网 AIGC 检测、
GPTZero 等）标记为 AI 生成内容。

逐句评估以下特征：
1. 句式是否过于均匀（每句字数差异<5字）
2. 是否存在"第一…第二…第三…"排比
3. 是否使用"此外/综上/由此可见"等模板连接词超过2次
4. 是否有无主语被动句
5. 段内是否有具体数据或引用支撑

如果发现 AI 痕迹，请标注具体位置并给出改写建议。

待检测段落：
```
