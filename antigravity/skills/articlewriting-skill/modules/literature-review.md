# 文献综述模块

本模块指导文献搜索、整理和综述写作，确保文献真实可靠。

---

## ⚠️ 核心原则：绝不编造文献

**这是最重要的原则，必须严格遵守：**

1. **英文文献**：可通过网络搜索获取，但必须验证真实性
2. **中文文献**：明确告知用户去知网（CNKI）搜索，AI提供搜索建议
3. **所有引用必须可追溯、可验证**
4. **不确定的文献信息，宁可不写也不编造**

## 一、文献搜索指南

### 1.1 英文文献搜索

**可用数据库（AI可搜索）：**

| 数据库 | 特点 | 适用领域 |
|--------|------|----------|
| Google Scholar | 综合性最强 | 全学科 |
| PubMed | 生物医学权威 | 医学、生物 |
| IEEE Xplore | 工程技术 | 计算机、电子 |
| arXiv | 预印本 | 物理、数学、CS |
| Semantic Scholar | AI增强搜索 | 全学科 |

**搜索策略：**

```
1. 确定核心关键词（英文）
2. 使用布尔运算符：AND, OR, NOT
3. 使用引号精确匹配："deep learning"
4. 限定时间范围：近5年优先
5. 按引用量排序找高影响力文献
```

**搜索示例：**
```
"transformer" AND "image classification" AND "attention mechanism"
```

### 1.2 中文文献搜索

**⚠️ AI无法直接搜索中文学术数据库，需要用户配合：**

**推荐数据库：**
- **知网（CNKI）**：最全面的中文学术数据库
- **万方数据**：学位论文丰富
- **维普**：期刊论文
- **百度学术**：综合搜索

**AI提供的帮助：**

1. **生成搜索关键词**
   ```
   用户研究主题：基于深度学习的图像分类
   
   建议搜索关键词：
   - 深度学习 + 图像分类
   - 卷积神经网络 + 图像识别
   - CNN + 特征提取
   - 迁移学习 + 图像
   ```

2. **提供搜索策略建议**
   ```
   1. 先搜索综述类文章，了解研究全貌
   2. 搜索高被引论文（知网可按被引排序）
   3. 搜索近3年文献，了解最新进展
   4. 搜索核心期刊/CSSCI来源期刊
   ```

3. **用户提供摘要后，AI帮助整理**
   ```
   用户：[粘贴知网搜索到的文献摘要]
   AI：整理文献信息，提取关键观点，纳入综述
   ```

## 二、文献整理

### 2.1 文献信息记录

**每篇文献记录以下信息：**

```markdown
## 文献1
- **标题**：
- **作者**：
- **年份**：
- **期刊/会议**：
- **DOI/链接**：
- **核心观点**：
- **研究方法**：
- **主要结论**：
- **与本研究关系**：
- **可引用点**：
```

### 2.2 文献分类

**按主题分类：**
```
文献综述/
├── 理论基础/
│   ├── 深度学习基础理论.md
│   └── 注意力机制理论.md
├── 方法技术/
│   ├── CNN相关方法.md
│   └── Transformer相关方法.md
├── 应用研究/
│   ├── 医学图像应用.md
│   └── 遥感图像应用.md
└── 综述文章/
    └── 领域综述汇总.md
```

### 2.3 文献评价

**评价维度：**

| 维度 | 评价要点 |
|------|----------|
| 相关性 | 与本研究主题的关联程度 |
| 权威性 | 期刊级别、作者影响力、被引次数 |
| 时效性 | 发表时间，是否反映最新进展 |
| 方法论 | 研究方法是否科学严谨 |
| 贡献度 | 对本研究的参考价值 |

## 三、综述写作

### 3.1 综述结构

**常用结构：**

```
一、引言
   - 研究背景
   - 综述目的和范围

二、研究现状
   2.1 主题一
       - 代表性研究
       - 主要观点
       - 研究进展
   2.2 主题二
       ...
   2.3 主题三
       ...

三、研究评述
   - 已有研究的贡献
   - 存在的不足
   - 研究趋势

四、研究空白与本研究定位
   - 尚未解决的问题
   - 本研究的切入点
```

### 3.2 写作要点

**综合而非罗列**
- ❌ 张三（2020）研究了A。李四（2021）研究了B。王五（2022）研究了C。
- ✅ 关于X问题，学界主要从三个角度展开研究：张三（2020）从A角度出发...；而李四（2021）则关注B方面...；王五（2022）进一步将研究拓展到C领域...

**批判性分析**
- 不仅介绍研究内容，还要评价其贡献和局限
- 指出不同研究之间的关系（支持、补充、矛盾）

**逻辑组织**
- 按主题而非按文献组织
- 从宏观到微观，从基础到前沿
- 最后聚焦到本研究的切入点

### 3.3 综述写作模板

**引入某一研究领域：**
```
[领域名称]是近年来[学科]研究的热点之一。自[开创性工作]以来，
该领域经历了快速发展，主要研究集中在[方向1]、[方向2]和[方向3]等方面。
```

**介绍代表性研究：**
```
[作者]（[年份]）提出了[方法/理论]，该研究[主要贡献]。
实验结果表明，[主要发现]。然而，该方法存在[局限性]。
```

**对比不同研究：**
```
与[研究A]不同，[研究B]采用了[不同方法]，在[某方面]取得了更好的效果。
然而，两者都未能解决[共同问题]。
```

**指出研究空白：**
```
尽管已有研究在[方面]取得了显著进展，但在[具体问题]方面仍存在不足。
具体而言，[问题1]尚未得到充分探讨，[问题2]缺乏系统性研究。
```

**引出本研究：**
```
基于上述分析，本研究拟从[角度]出发，针对[问题]，
提出[方法/观点]，以期[目标]。
```

## 四、引用管理

### 4.1 引用工具推荐

| 工具 | 特点 | 适用场景 |
|------|------|----------|
| Zotero | 免费开源，插件丰富 | 个人研究 |
| EndNote | 功能强大，高校常用 | 机构用户 |
| Mendeley | 社交功能，PDF管理 | 协作研究 |
| NoteExpress | 中文支持好 | 中文论文 |

### 4.2 引用格式

**中文（GB/T 7714-2015）：**
```
[1] 作者. 文章题目[J]. 期刊名, 年份, 卷(期): 页码.
[2] 作者. 书名[M]. 出版地: 出版社, 年份: 页码.
```

**英文（APA 7th）：**
```
Author, A. A., & Author, B. B. (Year). Title. Journal, Volume(Issue), pages. https://doi.org/xxx
```

**英文（IEEE）：**
```
[1] A. Author, "Title," Journal, vol. X, no. Y, pp. 1-10, Month Year.
```

### 4.3 引用检查

**必须检查：**
- [ ] 所有引用的文献都真实存在
- [ ] 作者、年份、标题信息准确
- [ ] 引用格式统一
- [ ] 正文引用与参考文献列表一一对应
- [ ] DOI/链接有效

### 4.4 真实性核验（深度校验）

当项目进入终稿阶段或用户明确要求核验参考文献时，调用 `modules/bibliography-verification.md` 执行系统化核验：

- 对全部文献执行三类分诊（真实且准确 / 真实但细节错 / 不存在或高风险伪造）
- 使用多源工具链（Crossref / AI4Scholar / 期刊往期目录等）逐条确认
- 确保替换文献后正文论述同步改写
- 最低证据标准：至少一个结构化来源命中，二手博客和搬运站不算确认

## 六、常见问题

### Q1：找不到足够的文献怎么办？

**解决方案：**
1. 扩大关键词范围，使用同义词
2. 放宽时间限制
3. 查找相关综述文章的参考文献
4. 使用引文追踪（被引/引用）
5. 考虑相邻领域的文献

### Q2：文献太多如何筛选？

**筛选标准：**
1. 优先选择高被引文献
2. 优先选择核心期刊/顶会
3. 优先选择近5年文献
4. 优先选择与研究主题直接相关的

### Q3：如何判断文献质量？

**评判指标：**
- 期刊影响因子/分区
- 文章被引次数
- 作者学术影响力
- 研究方法的严谨性
- 结论的可靠性
