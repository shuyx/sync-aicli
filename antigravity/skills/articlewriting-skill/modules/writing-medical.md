# 医学写作专项模块

本模块针对医学、生物、临床研究类论文的写作特点，提供专项指导。

---

## 一、医学论文特点

### 1.1 与其他学科的区别

| 方面 | 医学论文 | 一般理工科 |
|------|----------|------------|
| 结构 | 严格IMRAD + 报告规范 | IMRAD |
| 伦理 | 必须说明伦理审批 | 视情况 |
| 统计 | 要求详细统计方法 | 基本统计 |
| 报告 | 需遵循特定报告指南 | 无强制要求 |
| 注册 | 临床试验需预注册 | 无要求 |

### 1.2 常见医学论文类型

- **随机对照试验（RCT）**：药物/治疗效果评估
- **队列研究**：长期随访观察
- **病例对照研究**：回顾性分析
- **横断面研究**：某时点调查
- **系统综述/Meta分析**：文献综合分析
- **病例报告**：罕见病例描述

## 二、报告规范指南

### 2.1 EQUATOR Network 报告规范

**必须根据研究类型选择对应的报告规范：**

| 研究类型 | 报告规范 | 核心要求 |
|----------|----------|----------|
| 随机对照试验 | **CONSORT** | 流程图、随机化方法、盲法、ITT分析 |
| 观察性研究 | **STROBE** | 研究设计、参与者、变量、偏倚 |
| 系统综述 | **PRISMA** | 搜索策略、筛选流程、质量评估 |
| 诊断准确性 | **STARD** | 参考标准、盲法、2×2表 |
| 病例报告 | **CARE** | 时间线、诊断推理、随访 |
| 动物研究 | **ARRIVE** | 样本量、随机化、盲法 |
| 质量改进 | **SQUIRE** | 干预措施、背景、测量 |

### 2.2 CONSORT 核心清单（RCT）

**标题和摘要**
- [ ] 标题中标明"随机"
- [ ] 结构化摘要（目的、方法、结果、结论）

**引言**
- [ ] 科学背景和理论依据
- [ ] 具体目标或假设

**方法**
- [ ] 试验设计描述（平行、交叉等）
- [ ] 参与者纳入/排除标准
- [ ] 干预措施详细描述
- [ ] 预先确定的结局指标
- [ ] 样本量计算
- [ ] 随机化方法（序列生成、分配隐藏）
- [ ] 盲法实施
- [ ] 统计方法

**结果**
- [ ] 参与者流程图
- [ ] 基线特征表
- [ ] 每组结局数据（效应量+置信区间）
- [ ] 不良事件

**讨论**
- [ ] 局限性
- [ ] 可推广性
- [ ] 与其他研究比较

### 2.3 STROBE 核心清单（观察性研究）

**标题和摘要**
- [ ] 标题中标明研究设计
- [ ] 提供信息丰富的摘要

**方法**
- [ ] 研究设计关键要素
- [ ] 研究场所和时间
- [ ] 参与者（纳入标准、来源）
- [ ] 变量定义
- [ ] 数据来源/测量方法
- [ ] 偏倚控制措施
- [ ] 样本量说明
- [ ] 统计方法（混杂因素处理）

**结果**
- [ ] 参与者数量（流程图）
- [ ] 描述性数据
- [ ] 结局数据
- [ ] 主要分析结果

## 三、医学写作规范

### 3.1 伦理声明

**必须包含的内容：**

```
本研究经[医院名称]伦理委员会批准（批准号：XXX-XXX-XXX），
所有参与者均签署知情同意书。本研究遵循《赫尔辛基宣言》原则。
```

**临床试验注册：**
```
本试验已在中国临床试验注册中心注册（注册号：ChiCTR-XXX-XXXXXXXX）。
```

### 3.2 统计方法描述

**标准模板：**

```
统计分析使用SPSS 26.0（IBM Corp., Armonk, NY, USA）进行。
计量资料以均数±标准差（x̄±s）表示，组间比较采用独立样本t检验；
计数资料以例数（百分比）表示，组间比较采用χ²检验。
采用Kaplan-Meier法绘制生存曲线，组间比较采用Log-rank检验。
多因素分析采用Cox比例风险回归模型。
P<0.05为差异有统计学意义。
```

### 3.3 结果报告格式

**连续变量：**
- 正态分布：均数±标准差，如 `45.3±12.7岁`
- 非正态分布：中位数（四分位距），如 `23.5（15.2-38.7）`

**分类变量：**
- 例数（百分比），如 `156例（62.4%）`

**统计检验结果：**
- t检验：`t=2.45, P=0.016`
- 卡方检验：`χ²=8.32, P=0.004`
- 相关分析：`r=0.65, P<0.001`
- 回归分析：`OR=2.35, 95%CI: 1.42-3.89, P=0.001`

### 3.4 表格规范

**基线特征表示例：**

| 特征 | 试验组(n=50) | 对照组(n=50) | P值 |
|------|-------------|-------------|-----|
| 年龄(岁) | 45.3±12.7 | 44.8±11.9 | 0.842 |
| 男性, n(%) | 28(56.0) | 25(50.0) | 0.548 |
| BMI(kg/m²) | 24.2±3.1 | 23.8±2.9 | 0.498 |
| 病程(月) | 12(6-24) | 14(8-22) | 0.623 |

## 四、医学术语规范

### 4.1 常用缩写

| 全称 | 缩写 | 首次使用 |
|------|------|----------|
| randomized controlled trial | RCT | 首次出现时写全称 |
| confidence interval | CI | 95% confidence interval (95% CI) |
| odds ratio | OR | odds ratio (OR) |
| hazard ratio | HR | hazard ratio (HR) |
| body mass index | BMI | body mass index (BMI) |
| intention-to-treat | ITT | intention-to-treat (ITT) |

### 4.2 P值报告

- P<0.001：直接写 `P<0.001`
- 0.001≤P<0.05：写精确值，如 `P=0.023`
- P≥0.05：写精确值，如 `P=0.156`
- 避免写 `P=0.000`，应写 `P<0.001`

### 4.3 数字规范

- 小数点后保留位数一致
- 百分比保留1位小数
- P值保留3位小数
- 置信区间保留2位小数

## 六、医学论文检查清单

### 投稿前检查

**伦理与注册**
- [ ] 伦理批准号已注明

- [ ] 已选择正确的报告规范

**统计方法**
- [ ] 统计软件已注明
- [ ] 统计方法与数据类型匹配
- [ ] 样本量计算已说明
- [ ] 缺失数据处理已说明

**结果呈现**
- [ ] 效应量+置信区间
- [ ] P值格式正确
- [ ] 表格格式规范
- [ ] 图表清晰可读

**参考文献**
- [ ] 引用格式符合期刊要求
- [ ] 近5年文献占比充足
- [ ] 核心文献已引用
