# 阶段门禁（Stage Gates）

> 用途：把论文全流程拆成阶段，避免跳步导致返工。

## 项目

- 项目名称：
- 最后更新：
- 当前阶段：

## S0 启动与边界

- [ ] 已明确研究问题
- [ ] 已明确输出物
- [ ] 已明确截止日期
- [ ] 已记录写作偏好（无加粗、段间空一行、正文少列表）

## S1 方案与注册

- [ ] 已形成研究方案草稿
- [ ] 已判断是否需要预注册
- [ ] （如适用）已记录注册平台与模板

## S2 文献检索

- [ ] 已记录检索式和检索日期
- [ ] 已完成初步去重
- [ ] 已记录纳排标准

## S3 大纲与方法

- [ ] 已完成章节骨架
- [ ] 已对齐研究类型对应规范
- [ ] 已明确图表与数据计划

## S4 写作与图表

- [ ] 已完成核心章节初稿
- [ ] 已完成关键图表或脚本
- [ ] 已做一轮风格与排版检查

## S5 质量与合规

- [ ] 已做报告规范逐项核查
- [ ] 已做引用一致性检查
- [ ] 已检查伦理/利益冲突/AI使用披露要求

## S6 投稿包准备

- [ ] 已准备 cover letter
- [ ] 已准备作者贡献声明
- [ ] 已准备数据共享声明（如适用）
- [ ] 已完成期刊格式终检

## S7 返修与归档

- [ ] 已建立审稿意见响应矩阵
- [ ] 已完成版本变更记录
- [ ] 已归档关键复现材料
