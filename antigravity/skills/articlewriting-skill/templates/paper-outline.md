# 论文大纲生成指南

> **重要**：本文件不是固定模板，而是指导AI如何为用户动态生成论文大纲。

---

## AI工作流程

### 第一步：获取用户信息

与用户确认以下信息：
1. **学科领域**：具体学科（如：计算机科学、中国现当代文学、教育学、法学等）
2. **论文类型**：本科毕设/硕士论文/博士论文/期刊论文/课程论文
3. **目标期刊/学校**：如有具体要求
4. **研究主题**：论文题目或研究方向

### 第二步：搜索该类论文的标准结构

**必须执行**：使用WebSearch搜索该学科、该类型论文的标准结构。

搜索示例：
- "计算机科学硕士论文结构 章节安排"
- "中国现当代文学本科毕业论文结构"
- "教育学研究论文框架"
- "法学论文写作结构规范"
- "[目标期刊名] 论文结构要求"

### 第三步：生成定制化大纲

根据搜索结果，为用户生成符合其学科和论文类型的大纲，保存到 `plan/outline.md`。

## 常见学科结构参考

> 以下仅供参考，实际应以搜索结果为准。

### 理工科（IMRAD结构）
- 引言 → 文献综述 → 方法 → 实验/结果 → 讨论 → 结论

### 人文学科（论证结构）
- 引言 → 文献回顾 → 理论框架 → 文本/案例分析 → 讨论 → 结论

### 社会科学（研究设计结构）
- 引言 → 文献综述 → 研究设计 → 数据收集与分析 → 发现 → 讨论 → 结论

### 法学（案例分析结构）
- 引言 → 法律背景 → 案例分析 → 法理讨论 → 结论与建议

### 教育学（行动研究结构）
- 引言 → 文献综述 → 研究方法 → 实施过程 → 结果分析 → 反思与建议

### 历史学（史料分析结构）
- 引言 → 研究综述 → 史料考证 → 历史叙述与分析 → 结论

## 注意事项

1. **不要使用固定模板**：每个学科的论文结构不同，必须根据实际情况定制
2. **搜索验证**：对于不熟悉的学科，务必先搜索该学科的论文结构规范
3. **询问用户**：如果学校/期刊有特定格式要求，优先遵循
4. **持续更新**：大纲可能随研究进展调整，及时更新 `plan/outline.md`
